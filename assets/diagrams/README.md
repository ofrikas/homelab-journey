## Diagram Format

All diagrams use ASCII art and Mermaid syntax for easy version control and rendering in Markdown viewers.

To render Mermaid diagrams:
- GitHub automatically renders them
- VS Code: Install "Markdown Preview Mermaid Support" extension
- Online: https://mermaid.live/
